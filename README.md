# Juego de Gato (Tic Tac Toe)

## Descripción

Este programa implementa el juego de Gato (Tic Tac Toe) utilizando Python y la biblioteca Tkinter para la interfaz gráfica.

El juego permite dos modos de juego:

* Humano vs Humano.
* Humano vs Computadora.

Además, el programa detecta victorias por filas, columnas y diagonales, así como situaciones de empate. También cuenta con un botón para reiniciar la partida.

## Partes del código implementadas

Las siguientes partes fueron desarrolladas por mí:

* Creación de la interfaz gráfica con Tkinter.
* Generación del tablero de juego.
* Gestión de turnos entre jugadores.
* Verificación de ganador.
* Verificación de empate.
* Implementación del modo Humano vs Humano.
* Implementación del modo Humano vs Computadora.
* Botón para reiniciar la partida.
* Documentación básica del código.
